## Installation

### Swift Package Manager

Add Indice.Identity.iOS as a dependency
```swift
dependencies: [
    .package(url: "https://github.com/indice-co/Indice.Identity.iOS", .upToNextMajor(from: "1.0.0"))
]
```
Or you can clone the repo, and add it manually.
